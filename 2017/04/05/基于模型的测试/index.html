基于模型的测试

发表于 2017-04-05 | 分类于 软件测试

一个模型可以是对系统的任意描述。一个典型的行为模型会有一个开始状态，一个或多个转变，以及一个结束状态。

采用模型测试

  • 设计模型
  • 模型化软件
  • 建立有限状态模型

  • 模型自动化

  • 图论和MBT

  • 用于API测试的模型
  • 随机模型
  • 语法模型

不带测试的模型：

  • 贝叶斯图解模型
  • Petri网模型

任何类型的建模都是有益的，而随着模型而来的从模型产生的测试是十分强大的，模型能帮助测试工程师理解并解释复杂的系统，帮助管理风险，并且帮助寻找漏洞。当然基于模型的测试能做的不仅仅是在应用程序中随机行走，成功使用建模的团队也明白建模只是测试工具箱中的众多测试工具之一而已。
